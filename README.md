# PomodoroApp

A lightweight, clean desktop productivity timer, designed to help you stay focused and manage your work intervals effectively using the traditional Pomodoro technique.

---

## ⏳ How It Works

The application automatically manages the cycle between your focused work blocks, short breaks, and long breaks:

1. **Work Session (25 mins):** Click **Start** to begin your focus block. The app displays a "Work" badge, and the timer counts down.
2. **Short Break (5 mins):** After a work session completes, the app transitions into a short break to help you recharge.
3. **Long Break (15 mins):** Every time you complete **4 Pomodoro cycles**, the application awards you a longer 15-minute break to rest deeply before the next round.
4. **On-the-Fly Controls:** You can **Pause** the countdown at any point or use the **Reset button** (top right) to clear your current progress and restart the timer.
